# Perfumes Tendencia RD - Tienda de Perfumes Premium

Una aplicación web moderna y elegante para una tienda de perfumes premium, construida con Next.js, TypeScript, Tailwind CSS y Framer Motion.

## Características

- 🎨 Diseño premium y elegante
- 🔍 Búsqueda global en tiempo real
- 🎯 Sistema de filtros avanzado
- 🛒 Carrito de compras funcional
- 📱 Diseño completamente responsive
- ✨ Animaciones suaves con Framer Motion
- 🏷️ Catálogo organizado por marcas
- 💎 Experiencia de usuario inmersiva

## Tecnologías

- **Next.js 14** - Framework React con App Router
- **TypeScript** - Tipado estático
- **Tailwind CSS** - Estilos utility-first
- **Framer Motion** - Animaciones fluidas
- **Lucide React** - Iconos modernos

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Estructura del Proyecto

```
├── app/              # App Router de Next.js
├── components/       # Componentes reutilizables
├── data/            # Datos mock
├── types/           # Tipos TypeScript
├── hooks/           # Custom hooks
└── lib/             # Utilidades
```
